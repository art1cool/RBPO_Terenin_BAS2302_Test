# Музыкальный стриминговый сервис
Используемые сущности: Музыкант, Альбом, Трек, Пользователь, Плейлист

Музыкант имеет атрибуты: имя, жанр

Альбом: название, год выхода, музыкант

Трек: название, музыкант

Пользователь: имя

Плейлист: название, пользователь

У музыканта будут альбомы и треки без альбомов, у пользователя - плейлисты
